CSS

CSS ещё проще, чем HTML. Он состоит из
множества правил, примерно таких:

селектор {
      свойство1: значение1;
      свойство2: значение2;
}

Правила очень простые. Но есть одна проблема:
свойств очень много.

Большая часть курса будет посвящена именно CSS.

А сейчас поработайте с нижним редактором.
